# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by:

1. **Do not** open a public GitHub issue
2. Email `j6n.work@gmail.com` with:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested fixes (if applicable)

We aim to respond within 48 hours and will work with you to coordinate a disclosure and fix.

## Security Updates

Security updates will be released as patch versions and announced without details until the fix is available.
